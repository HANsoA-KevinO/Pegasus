# 经济学模型图表规范

## 博弈论图

### 策略式博弈（矩阵形式）
- 使用 2×2 或 n×m 矩阵表格
- 行玩家和列玩家分别标注
- 纳什均衡格用加粗边框或底色高亮
- 最优反应用箭头或下划线标注

### 扩展式博弈（博弈树）
- 决策节点：实心圆点 + 玩家标签
- 终端节点：支付向量
- 信息集：同一玩家的节点用虚线连接
- 概率节点：空心圆点 + Nature 标签

### 反应函数
- 两个玩家的最优反应函数在同一坐标系
- 交点为纳什均衡，用大圆点标注
- 动态调整路径用箭头标注

## 金融模型图

### 蒙特卡洛模拟路径
- 多条随机路径用半透明线条
- 均值路径用粗实线
- 置信区间用阴影带
- 时间轴标注清晰

### 期权定价
- 标的资产价格树用二叉树结构
- 看涨/看跌用不同颜色
- 执行价用水平虚线标注

## 宏观经济模型

### IS-LM
- IS 曲线负斜率，LM 曲线正斜率
- 横轴 Y (产出)，纵轴 r (利率)
- 政策效果用曲线移动 + 箭头表示

### AD-AS
- AD 负斜率，SRAS 正斜率，LRAS 垂直
- 短期/长期均衡分别标注
