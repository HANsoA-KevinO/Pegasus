# 经济学领域绘图规范

## 常见图表类型

- **博弈论图**：博弈树、反应函数、纳什均衡
- **供需图**：供给/需求曲线、均衡点、消费者/生产者剩余
- **金融模型图**：资产定价路径、风险收益图、期权定价
- **宏观模型图**：IS-LM、AD-AS、菲利普斯曲线
- **计量模型图**：因果推断 DAG、面板数据结构

## 视觉惯例

### 坐标系图
- 坐标轴清晰标注变量名和单位
- 曲线用不同颜色和线型区分
- 均衡点用大圆点标注
- 移动方向用箭头标注

### 博弈论
- 博弈树从上到下展开
- 决策节点用实心圆点
- 信息集用虚线连接
- 支付矩阵用表格形式

### 因果推断
- DAG 中变量用圆形节点
- 因果关系用单向箭头
- 混杂变量用不同颜色标注
- 处理变量和结果变量突出显示
