# 神经网络架构图规范

## Transformer 架构
- Encoder/Decoder 用大的容器框，内含多个子层
- Multi-Head Attention 用多个平行箭头表示
- Feed-Forward 用单个矩形
- 残差连接用弧线绕过子层
- Layer Norm 用细条形矩形
- 位置编码用加号图标

## CNN 架构
- 卷积层用 3D 方块（高度=通道数，宽度=空间大小）
- 池化用缩小的方块
- 特征图用颜色渐变表示激活

## GAN 架构
- Generator 和 Discriminator 分左右或上下
- 潜空间用随机噪声图标 (z)
- 真实/生成数据流用不同颜色区分
- 对抗训练用循环箭头

## 通用规范
- 维度标注：在层旁用小字标注 (B, C, H, W) 或 (seq_len, d_model)
- 数学符号保留 LaTeX 格式
- 数据流方向保持一致（通常从左到右或从下到上）
- 颜色编码按功能分组（注意力=蓝色系，前馈=绿色系 等）
