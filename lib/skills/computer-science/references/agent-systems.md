# 多智能体系统图表规范

## 核心元素

### Agent 表示
- 每个 Agent 用带圆角的矩形框，内含图标和名称
- 不同角色的 Agent 使用不同颜色区分
- Agent 内部子模块用缩进的小矩形表示

### 通信与交互
- 同步通信：实线双向箭头
- 异步通信：虚线单向箭头
- 广播：一对多扇形箭头
- 消息内容用箭头上的标签说明

### 共享资源
- 共享记忆：圆柱体或云形图标
- 工具/API：六边形或齿轮图标
- 环境：包围所有 Agent 的大虚线框

### 流程编排
- 时间顺序用数字标注
- 并行执行用并排布局
- 条件分支用菱形决策节点

## 布局建议
- 主控 Agent 居中或居上
- 工作 Agent 环绕或并排
- 共享资源居中下方或侧边
- 留足间距，避免箭头交叉
