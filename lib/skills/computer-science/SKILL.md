# 计算机科学领域绘图规范

## 常见图表类型

- **模型架构图**：Transformer、CNN、GAN 等网络结构
- **系统架构图**：多模块系统、微服务、管道流程
- **多智能体系统图**：Agent 交互、通信拓扑、共享记忆
- **算法流程图**：训练循环、推理管道、数据处理流

## 视觉惯例

### 神经网络架构
- 层用矩形或圆角矩形表示，堆叠排列
- 注意力机制用箭头矩阵或热力图表示
- 数据流从左到右或从下到上
- 维度标注使用 LaTeX 格式

### 多智能体系统
- Agent 用拟人化图标或带标签的圆角框
- 通信用双向箭头，共享资源用云形或数据库图标
- 环境用大的背景容器框

### 系统架构
- 模块用矩形，按功能分层
- API 边界用虚线分隔
- 数据流用实线箭头，控制流用虚线箭头
